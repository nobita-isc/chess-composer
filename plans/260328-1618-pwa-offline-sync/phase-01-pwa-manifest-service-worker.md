## Implementation Steps

### 1. Install vite-plugin-pwa
```bash
npm install -D vite-plugin-pwa -w @chess-composer/client
```

### 2. Create manifest.json
```json
{
  "name": "Chess Trainer",
  "short_name": "Chess Trainer",
  "description": "Chess puzzle trainer for students",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#4f46e5",
  "icons": [
    { "src": "/favicon.svg", "sizes": "any", "type": "image/svg+xml" }
  ]
}
```

### 3. Configure VitePWA in vite.config.js
```javascript
import { VitePWA } from 'vite-plugin-pwa'

export default defineConfig({
  plugins: [
    VitePWA({
      registerType: 'autoUpdate',
      workbox: {
        globPatterns: ['**/*.{js,css,html,svg}'],
        runtimeCaching: [
          {
            urlPattern: /^https:\/\/fonts\.googleapis\.com/,
            handler: 'StaleWhileRevalidate'
          },
          {
            urlPattern: /^https:\/\/fonts\.gstatic\.com/,
            handler: 'CacheFirst',
            options: { cacheName: 'google-fonts', expiration: { maxEntries: 10 } }
          }
        ]
      },
      manifest: false, // use public/manifest.json
      includeAssets: ['favicon.svg']
    })
  ]
})
```

### 4. Update index.html
- Add `<link rel="manifest" href="/manifest.json">`
- Add `<meta name="theme-color" content="#4f46e5">`
- Add `<meta name="apple-mobile-web-app-capable" content="yes">`

### 5. Generate PWA icons (optional)
- 192x192 and 512x512 PNG icons from the SVG favicon
- Add to manifest.json icons array

## Cache Strategy
- **App shell** (HTML, CSS, JS, SVG): Precached by Workbox on install
- **Google Fonts**: StaleWhileRevalidate (fast load, update in background)
- **API calls**: Network-only (NOT cached — always live data)
- **Chess piece images**: CacheFirst (static assets, never change)

## Success Criteria
- [ ] App installable on Chrome/Safari mobile
- [ ] App loads offline (shell only, no API data)
- [ ] Lighthouse PWA audit passes
- [ ] Service worker registers and activates
